üì∏ Cara Membuat OG Image (1200x630px)

  1. Buka file HTML ini di browser (Chrome/Edge recommended)
  2. Tekan F12 untuk membuka DevTools
  3. Tekan Ctrl+Shift+P (Windows) atau Cmd+Shift+P (Mac)
  4. Ketik "screenshot" dan pilih "Capture node screenshot"
  5. Klik pada element dengan class og-container di DevTools
  6. Screenshot akan otomatis terdownload
  7. Rename file menjadi og-image.jpg
  8. Pindahkan ke folder public/ di project Next.js Anda

Alternatif: Gunakan tools online seperti Canva atau Figma dengan ukuran canvas 1200x630px dan desain yang sama.
